# 🏴‍☠️ Visual Novel: Ship Deck

Реализация визуальной новеллы в фентези сеттинге на C++ с использованием графической библиотеки SDL3 и модулей расширения.

## Описание

Демо-версия первого акта на палубе корабля (пролог и диалоги урезаны). Игрок управляет персонажем Имотэсом, перемещаясь по палубе с учетом физических границ и препятствий (мачта, бочки, якорь). Реализована интерактивная система диалогов: при клике мышкой на NPC (например, юнгу Марко) игра переходит в режим сюжетной паузы, фон кинематографично затемняется, а справа выводится портрет персонажа с ветвлением реплик на русском языке.

## Особенности сборки (Важно)

В отличие от стандартных сборок, данный проект полностью **автономен**. Все необходимые SDK (заголовки, библиотеки линковки и динамические `.dll`) уже включены в репозиторий в папках `SDL/`, `SDL_image/` и `SDL_ttf/`. 

Это гарантирует успешную компиляцию на **абсолютно чистой системе** без необходимости предварительной установки библиотек в операционную систему.

## Требования

Для сборки проекта необходимо, чтобы в системе были установлены:
- Bash-совместимый терминал (например, [**Git Bash**](https://gitforwindows.org) или [**MSYS2**](https://www.msys2.org/))
- Компилятор **GCC / G++** с поддержкой C++17
- **GNU Make** (или `mingw32-make`)

## Сборка и запуск

Склонируйте репозиторий и перейдите в его папку:

```bash
git clone https://github.com
cd VisualNovel
```

### Вариант 1. Сборка через Makefile (Рекомендуется)
Запустите сборку одной командой из терминала:
```bash
make
```
*(Или `mingw32-make`, если утилита в MSYS2 называется полным именем).* 

Команда автоматически скомпилирует исходники в папку `bin/`, скопирует туда текстуры, шрифты и перенесёт все три системные `.dll`.

Запуск игры:
```bash
make run
```

### Вариант 2. Сборка через CMake
Проект сохраняет полную кроссплатформенность, конфигурационный файл `CMakeLists.txt` доступен для сборки в любой IDE (VS Code, CLion, Visual Studio):
```bash
cd build
cmake ..
cmake --build .
```

## Генерация документации

Проект использует Doxygen для автоматической генерации интерактивной документации из комментариев в коде. 

Выполните команду:
```bash
make docs
```
После этого откройте файл `index.html` из созданной папки `docs/html/` в любом браузере для просмотра структуры классов и методов.
